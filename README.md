# 🌍 Travelia - Explore the World

Travelia is a modern, responsive, and visually stunning travel agency website template built using semantic HTML5, custom CSS properties, and lightweight vanilla JavaScript. It is designed to offer potential travelers an effortless experience while browsing top destinations, premium service values, and authentic trip statistics.

# Features

* Preloader Hiding:** A seamless animated SVG preloader that smoothly transitions out once the window is completely loaded.
* Mobile-First Responsive Navigation:** A clean mobile toggle menu with an integrated backdrop blur overlay for modern UX.
* Sticky Header:** Dynamic navigation state changes automatically once the user scrolls past 100px.
* Performance Optimized:** Preloaded hero banners and background images minimize layout shifts (CLS).
* Flexible Typography & Theming:** Built entirely on CSS Custom Properties (`:root`) enabling effortless rebranding and colorshift implementations.

---

## 🛠️ Tech Stack & Dependencies

* Markup:** HTML5 (Semantic Structure)
* Styling:** CSS3 (Custom Properties & Flexbox/Grid architecture)
* Interactivity:** Vanilla JavaScript (ES6 Strict Mode)
* Icons:** [IonIcons](https://ionic.io/ionicons) (`v5.5.2` ESM & No-Module bundles)
* Fonts:** Google Fonts (Poppins family: 400, 500, 600, 700)
